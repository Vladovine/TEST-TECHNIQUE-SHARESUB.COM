# Risk Monitor

Cas pratique Sharesub — outil interne de détection et traitement des abonnés à risque.

## Objectif
Construire un outil avec :
- exploration et nettoyage de données SQLite
- scoring de risque
- interface web
- agent IA
- documentation claire

## Structure
- `app/` : interface web
- `data/raw/` : base SQLite brute
- `data/processed/` : données traitées
- `notebooks/` : notebook d’exploration
- `outputs/` : exports
- `prompts/` : prompts IA versionnés
- `src/` : code Python réutilisable
