# Prompts agent IA

Les prompts sont versionnés pour garder la traçabilité des itérations.

- `agent_analyst_v1.txt` : résumé structuré du cas côté analyse.
- `agent_decider_v1.txt` : recommandation d’action côté opérationnel.

Les sorties attendues sont en JSON valide pour faciliter l’intégration dans l’interface.
